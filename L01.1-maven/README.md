# L01.1-maven

Toy project with maven dependencies.

How to run:
* run `$ mvn clean package`
* and then, run `$ java -jar target/L01.1-maven-1.0.0.jar`
* or, you can use docker `$ docker run --rm otusjava201804lobanov/l011-maven`
